# CTC (Coaching Training Course)

## 概要

**CTC** は、CBS で習得した基礎的なコーチング理論とスキルを、実務場面での実践へと発展させるコースです。実際のコーチング場面での複雑な課題への対応、クライアント支援の深化、継続的なコーチング関係の構築方法を習得する **実践トレーニングコース** です。

## コースの目的

コーチングを理論から実務へ、個別事例から組織的実践へと発展させることが目標です。

## 構成（推定）

### 第一段階：基礎の再確認と応用準備（セッション 1-2）
理論と実践のギャップを埋め、実務ベースでの対応準備

### 第二段階：実践的スキルの深化（セッション 3-5）
複雑なコーチング場面への対応、深い気づきの促進、行動変化の定着

### 第三段階：継続的コーチング関係の設計（セッション 6-8）
複数セッションプログラムの設計、関係の継続と深化、成果測定

### 第四段階：組織的コーチング実践（セッション 9-10）
組織内コーチング文化の構築、継続的専門性開発

## コース情報

| 項目 | 内容 |
|-----|------|
| **シリーズ** | 10 セッション（推定） |
| **学習時間** | 20～30 時間 |
| **推奨期間** | 2～3 ヶ月（週 1～2 回） |
| **対象者** | コーチング実践者、人材育成担当者 |
| **前置知識** | CBS 修了（必須）、実践経験 |
| **推奨順序** | CBS 修了後 1～3 ヶ月以内 |

## 関連コース

- **コミュカUP**: 基本的なコミュニケーション能力の強化
