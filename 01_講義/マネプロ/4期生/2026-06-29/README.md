# マネプロ 4期生 講義ノート

## シリーズ概要

**プログラム名：** マネプロ（Money Program）4期生  
**開催日：** 2026-06-29  
**主催：** awareness  
**URL：** https://www.awareness.co.jp/money-program/term-seminar-list

マネプロは「お金の知識を実践に変える」ことを目的とした全10回の体系的な金融教育プログラムです。家計管理から投資・税金・ライフプランまで、経済的自由を実現するための一貫した学びを提供します。

## カリキュラムの構成

```
基礎固め（01〜03）
  ├── お金の本質・マインドセット
  ├── 家計の現状把握
  └── 先取り貯蓄の仕組み化

資産形成（04〜06）
  ├── 投資の基本（インデックス投資・NISA・iDeCo）
  ├── 株式投資（長期・分散・積立）
  └── 不動産投資（実物資産・REIT）

知識の深化（07〜08）
  ├── 税金と保険（節税・保険の最適化）
  └── ライフプラン設計（キャッシュフロー表・老後資金）

統合・目標設定（09）
  └── 経済的自由（FIRE・豊かな人生の実現）
```
