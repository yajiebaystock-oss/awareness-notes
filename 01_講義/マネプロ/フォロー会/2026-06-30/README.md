# マネプロ フォロー会 2026-06-30

## 概要

マネプロ（マネープログラム）のフォロー会は、マネプロ受講生が実際に投資・資産形成の実践を進めながら、疑問や課題をQ&A形式で解消していく継続学習セッションです。全8回のセッションで構成され、投資基礎から家計改善・NISA・iDeCo・株式投資・不動産投資・税金・保険・ライフプランまで幅広いテーマをカバーします。

受講生が自らの体験・実践報告を共有することで、学び合いの場を提供します。

## 各セッションの主要テーマ

### 第1回：投資基礎のQ&A・実践報告
- 複利の仕組みと計算実習
- 分散投資の重要性
- インデックスファンドvs アクティブファンド
- 信託報酬（手数料）の長期的インパクト
- リスク許容度の自己診断
- 長期・分散・積立の3原則

### 第2回：家計改善の進捗確認・疑問解消
- 固定費（通信費・保険料）の削減実践
- 先取り貯蓄・先取り投資の自動化
- 支出の3分類（固定費・変動費・特別支出）
- サブスクリプションの棚卸し
- 貯蓄率を家計健全度の指標にする方法

### 第3回：NISA・iDeCo実践Q&A
- 新NISAの年間投資枠と生涯投資枠の活用
- つみたて投資枠と成長投資枠の使い分け
- iDeCoの所得控除による節税効果の計算
- iDeCoの商品選択と金融機関の選び方
- NISAとiDeCoの優先順位の考え方

### 第4回：株式投資の実践報告と課題
- 個別株vs投資信託のリスク・リターン比較
- コア・サテライト戦略によるポートフォリオ管理
- SNS情報の危険性と自分で調べることの重要性
- アセットアロケーションの考え方
- 損切りルールと損失回避バイアスへの対処

### 第5回：不動産投資Q&A・事例共有
- 表面利回りと実質利回りの計算と違い
- 立地重視の物件選びの原則
- 融資（レバレッジ）のメリットとリスク
- 区分マンション・一棟アパート・REITの比較
- 出口戦略から逆算する物件選びの視点

### 第6回：税金・保険最適化の実践報告
- ふるさと納税の活用方法と限度額の計算
- 高額療養費制度を理解した上での保険見直し
- 年末調整で漏れやすい控除の確認
- 副業収入の確定申告と青色申告の活用
- ライフイベントに合わせた保険の定期見直し

### 第7回：ライフプラン見直しQ&A
- 教育費の費用感と早期準備の重要性
- 老後2,000万円問題の本質と個人の必要額計算
- マイホーム購入の判断基準と住宅ローンの目安
- キャッシュフロー表の作成方法
- 3つの財布（日常費・教育費・老後資金）の分別管理

### 第8回：総まとめ・次のステップ
- 8回を通じた7つの行動変容の振り返り
- 資産形成の4本柱（稼ぐ・節約・増やす・守る）
- 複利思考の人生全体への応用
- 積立投資の自動化・メンテナンスフリー化
- 財務的自由（FIRE）への具体的ロードマップ
- 5年後の財務目標と今月から始める3つのアクション
