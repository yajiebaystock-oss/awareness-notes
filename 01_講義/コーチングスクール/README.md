# コーチングスクール

## 概要

Awareness Vault のコーチングスクールは、より高度なコーチング理論と実践スキルを習得するための講義シリーズです。ベーシックスクール修了者向けの上級プログラムとなっています。

## 含まれるシリーズ

### AMS（Advanced Management Series）
- **テーマ**: 上級マネジメントシリーズ
- **対象**: 上級者・管理職
- **内容**: リーダーシップと戦略的マネジメント

### EQS（Emotional Quotient Series）
- **テーマ**: 感情知能スキルシリーズ
- **対象**: 中級者以上
- **内容**: 感情知能の深化と応用、自己管理と対人関係のスキル向上

## 学習の流れ

コーチングスクールはベーシックスクールの基礎を応用した、より実践的で高度なスキル習得を目指しています。
