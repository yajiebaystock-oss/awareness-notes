# BPC Advance (Priority Coaching Advance)

## 概要

**BPC Advance** は、Awareness の独自の「優先度（Priority）」思考体系を活用したビジネスコーチング応用コースです。個人の成長、組織の発展、そしてビジネスの成功という複数の目標を、優先度思考によって統合し、実務的かつ戦略的なコーチング実践を習得する、最も実務的で応用度の高いコースです。

## コースの目的

ビジネスコンテキストで、個人の成長と組織のビジネス成果を同時に実現するコーチング実践を習得することが目標です。

## 構成（推定）

### 第一段階：ビジネスコーチングの基礎（セッション 1-2）
個人向けコーチングと組織向けビジネスコーチングの違い、優先度思考の統合

### 第二段階：ビジネス課題のコーチング的解決（セッション 3-5）
売上・ビジネス成果の達成、個人成長との両立、組織間の調整

### 第三段階：経営的視点のコーチング（セッション 6-8）
リーダーシップ育成、組織変革、組織的コーチング戦略の設計・実装

### 第四段階：ビジネスコーチングの統合実践（セッション 9-10）
複合的ビジネス課題への対応、組織への貢献

## 特色

### Awareness 優先度思考との統合

BPC Advance は、Awareness の「優先度（Priority）」思考体系を特に重視するコースです。これにより：

- 個人の成長目標と組織のビジネス目標を統一的に考える
- 複数の目標の整合性を取りながら実行する
- 個人と組織の成長を同時に実現する

このアプローチが、BPC Advance をコーチングスクール内で最も実務的で、ビジネス成果に直結するコースとしています。

### 実務的・戦略的

BPC Advance は、単なるコーチング技法ではなく、ビジネス戦略とコーチングの統合を実現するコースです。
