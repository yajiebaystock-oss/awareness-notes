# SLS (Supportive Leadership Series)

## 概要

**SLS** は、コーチングスクール内でリーダーシップとコミュニケーションを統合した発展シリーズです。CBS で習得した基本的なコーチングスキルを、リーダーシップ視点から応用・深化させる **16 セッションシリーズ** です。

## コースの目的

組織やチームにおいて、部下やチームメンバーのポテンシャルを引き出し、個人と組織の成長を同時に推進するリーダーシップの実践方法を習得することが目標です。

## 構成

### 自己認識と基盤構築段階（セッション 1-2）
リーダーシップの基礎となる自己認識と戦略立案

### コミュニケーションと関係構築段階（セッション 3-4）
リーダーシップコンテキストでの対話と変化対応

### ストレス管理と意思決定段階（セッション 5-6）
心身の健全性と戦略的判断力

### チーム構築と協働段階（セッション 7-9）
高機能なチームの構築と課題解決

### 発展と継続学習段階（セッション 10-16）
組織規模でのリーダーシップ実践と倫理的リーダーシップの確立

## コース情報

| 項目 | 内容 |
|-----|------|
| **シリーズ** | 16 セッション |
| **学習時間** | 32～48 時間 |
| **推奨期間** | 3～4 ヶ月（週 1～2 回） |
| **対象者** | リーダー・マネージャー、リーダーシップ志向者 |
| **前置知識** | CBS 修了を推奨 |
| **リーダーシップ経験** | 実際にチームをマネジメントした経験があると学習効果が高い |
| **推奨順序** | CBS 修了後、またはプレミアムコース並行受講 |

## 関連コース

### 前置シリーズ（推奨）
- **CBS** (Coaching Basic Skills): 傾聴、質問、フィードバックなどの基本スキル

### 同時受講推奨
- **コミュカUP**: 実践的なコミュニケーション能力の強化
- **BPC Advance**: ビジネス視点からのコーチング応用

### 発展シリーズ
- **CTC** (Coaching Training Course): コーチング実践の一層の深化
- **CTE** (Coaching Training Executive): エグゼクティブレベルのリーダーシップ実践
- **Premium Course**: 認定コーチプログラムへの進学

## 推奨学習プロセス

```
1. 第一段階：リーダーシップ基礎の確立（セッション 1-2）
   → リーダー自身の自己理解と戦略立案

2. 第二段階：コミュニケーションと変化対応（セッション 3-4）
   → 対話の質と変化への適応力を向上

3. 第三段階：チーム管理とリーダーシップ基本スキル（セッション 5-9）
   → ストレス管理、意思決定、チーム構築を習得

4. 第四段階：組織規模でのリーダーシップ実践（セッション 10-16）
   → 倫理的で変革的なリーダーシップを確立
   → グローバル対応とリーダーシップスタイルの統合
```

## 学習効果の測定

- 各セッション後の理解度チェック
- リーダーシップスキル自己評価
- 360 度フィードバック（部下・同僚・上司）
- ビジネス成果の測定（チームパフォーマンス、エンゲージメント）

## コース修了後

### 修了認定基準
- 16 セッション全てを受講完了
- グループディスカッションへの積極的参加
- 最終リーダーシッププラン発表の完了

### 継続学習へのパス
- CTE などのエグゼクティブ・プログラムへの進学
- 職場でのリーダーシップ実践の継続
- 認定取得を目指す Premium Course への進学検討
- リーダーシップコミュニティでの継続参加

## 期待される成果

このコース修了時に、以下の能力が身につきます：

- リーダー自身の自己認識と継続的成長能力
- 戦略的思考と意思決定能力
- 効果的なコミュニケーション能力
- チーム構築とメンバー育成能力
- 組織の変化と成長推進能力
- 倫理的で責任あるリーダーシップ実践能力
- グローバルコンテキストでの対応能力

## CBS との関連性

SLS は CBS の発展シリーズとして位置付けられています：

| スキル | CBS での習得 | SLS での発展 |
|-------|------------|-----------|
| 傾聴 | 個人レベルのアクティブリスニング | 組織内の複数レベルでのコミュニケーション |
| 質問 | 相手の思考を刺激する質問 | 戦略的な思考を刺激する質問 |
| フィードバック | 個人への建設的フィードバック | 組織全体への情報発信と指示 |
| 関係構築 | 個人レベルでの信頼構築 | チームレベルでの信頼文化構築 |
