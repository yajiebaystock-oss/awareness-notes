# CBS (Coaching Basic Skills)

## 概要

**CBS** は、コーチングスクール内で最も基本的で重要なシリーズです。コーチングの理論から実践スキルまで、段階的かつ体系的に習得できるように設計されている **18 回シリーズ** です。

## コースの目的

相手の潜在能力を引き出し、目標達成をサポートするための思考様式とスキルを習得することが目標です。

## 構成

### 基礎概論段階（エピソード 1-2）
コーチングの本質と理論的背景を理解

### 関係構築段階（エピソード 3-5）
信頼と心理的安全性に基づくコーチング関係を構築

### スキル習得段階（エピソード 6-11）
傾聴、質問、フィードバック、目標設定などの基本技術を習得

### 応用実践段階（エピソード 12-18）
複雑な状況への対応と倫理的なコーチング実践を習得

## コース情報

| 項目 | 内容 |
|-----|------|
| **シリーズ** | 18 回エピソード |
| **学習時間** | 18～36 時間 |
| **推奨期間** | 3～6 ヶ月（週 1 回） |
| **対象者** | 全レベル（初心者から経験者まで） |
| **前置知識** | なし |
| **推奨順序** | コーチングスクール入学直後 |

## 推奨学習プロセス

```
1. 第一段階：理論基盤の構築（エピソード 1-2）
   → コーチングの本質と姿勢を理解

2. 第二段階：関係構築スキルの習得（エピソード 3-5）
   → 信頼関係を構築

3. 第三段階：基本技術スキルの習得（エピソード 6-11）
   → 傾聴、質問、フィードバックを習得

4. 第四段階：応用と統合（エピソード 12-18）
   → 複雑な状況に対応でき、倫理的なコーチング実践を確立
```

## 学習効果の測定

- 各エピソード後の理解度チェック
- コーチングスキルの自己評価
- 実務での実践報告
- 関係者からのフィードバック

## コース修了後

### 修了認定基準
- 18 エピソード全てを受講完了
- 実践演習への積極的参加
- 最終成果発表の完了

### 継続学習へのパス
- SLS などの発展コースへの進学
- 認定取得を目指す Premium Course への進学検討
- コミュニティでの継続参加
