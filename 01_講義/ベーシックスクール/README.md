# ベーシックスクール

## 概要

Awareness Vault のベーシックスクールは、コーチング・ファシリテーションの基礎を学ぶための講義シリーズです。

## 含まれるシリーズ

### T2Sセミナー
- **テーマ**: 2つのシステムとセルフコーチング
- **対象**: コーチング初心者から中級者
- **内容**: システム1・システム2の理論と実践的活用

### SLS（Supportive Leadership Series）
- **テーマ**: サポーティブリーダーシップシリーズ
- **対象**: 全レベル
- **内容**: 他者をサポートするためのリーダーシップスキル

### CPS（Coaching Problem Solving）
- **テーマ**: コーチングによる問題解決
- **対象**: 中級者
- **内容**: クライアントの問題解決をサポートするコーチング手法

### HTP（Humanistic Transactional Process）
- **テーマ**: 人文的交流プロセス
- **対象**: 全レベル
- **内容**: 人間関係構築のための基本プロセス

### ICF（Intentional Coaching Framework）
- **テーマ**: 意図的コーチングフレームワーク
- **対象**: 初級者から中級者
- **内容**: 構造的コーチングアプローチの実装

### IES（Integrated Emotional System）
- **テーマ**: 統合感情システム
- **対象**: 全レベル
- **内容**: 感情知能とコーチングの統合

### CBS（Core Belief System）
- **テーマ**: コアビリーフシステム
- **対象**: 全レベル
- **内容**: 信念体系の理解と変容

### ECS（Effective Communication Series）
- **テーマ**: 効果的コミュニケーションシリーズ
- **対象**: 全レベル
- **内容**: コーチング場面での効果的なコミュニケーション

### AMS（Advanced Management Series）
- **テーマ**: 上級マネジメントシリーズ
- **対象**: 上級者・管理職
- **内容**: リーダーシップと戦略的マネジメント

### EQS（Emotional Quotient Series）
- **テーマ**: 感情知能スキルシリーズ
- **対象**: 中級者以上
- **内容**: 感情知能の深化と応用、自己管理と対人関係のスキル向上

## 学習の流れ

ベーシックスクールは、基礎的な理論から実践的スキルへと段階的に進む設計となっています。初心者は T2Sセミナー から始めることをお勧めします。
