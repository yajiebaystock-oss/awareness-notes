# CBS（コーチングの基礎）ガイド

## 概要

CBS（Coaching Basic Series）は、コーチングの本質から実践的スキル、倫理的基準まで、システマティックに学べる18回の統合的学習プログラムです。相手の潜在能力を引き出し、目標達成をサポートするための知識とスキルを体系的に習得できます。

このディレクトリには、複数年度版のCBSコンテンツが版ごとに整理されています。

## セッション構成（2026-06-27版）

### 基礎層（1-6）：コーチング基本知識と自己認識

1. **コーチングの本質と定義** - コーチングとは何か、他の支援形態との違いを理解
2. **コーチング思考と心構え** - コーチとして必要な信念体系とマインドセット
3. **自己認識とコーチ自身の成長** - メタ認知とバイアス認識を通じた自己理解
4. **クライアント理解の基本** - ニーズ把握、現状分析、目標設定支援
5. **コーチング関係の構築** - 心理的安全性と信頼醸成
6. **傾聴スキルの基礎** - 受動的・能動的傾聴と反射技法

### スキル層（7-12）：コーチングスキル習得

7. **質問力の段階的展開** - 開かれた質問、探索的質問、深掘り質問
8. **フィードバック技法** - 観察に基づく効果的なフィードバック方法
9. **目標設定と行動計画** - SMART原則に基づいた目標策定
10. **モチベーション管理支援** - 内発的動機づけと持続的な行動支援
11. **セッション構造と流れ** - オープニング、メインテーマ、アクションプラン
12. **多様なコーチング手法** - メタファ、シナリオ分析、リソース活用型アプローチ

### 実践層（13-18）：対人コーチング応用

13. **困難なクライアント対応** - 抵抗や感情的反応への対応
14. **変化促進とサポート** - 行動変容プロセスと習慣化支援
15. **コーチング倫理と境界線** - 倫理基準と専門的関係の維持
16. **複数セッションの設計** - 長期的な成長計画と段階的実施
17. **コーチングの成果測定** - 定量的・定性的な評価方法
18. **継続的専門性開発と統合** - コーチとしての生涯学習とコミュニティ

## 学習方法

### 段階的学習

1. **基礎層から開始**（セッション1-6）
   - コーチングの本質と自分自身の理解
   - コーチングマインドセットの醸成
   - 所要時間：2-3週間

2. **スキル習得**（セッション7-12）
   - 実践的なコーチングテクニックの習得
   - 各スキルの段階的深掘り
   - 所要時間：4-6週間

3. **実践応用**（セッション13-18）
   - 複合的な場面での対応力の構築
   - コーチング実践の質的向上
   - 倫理基準と専門性の確立
   - 所要時間：3-4週間

### 推奨学習時間
- **全セッション完了**: 9-13週間（約2～3ヶ月）
- **セッションあたり**: 2-3時間（講義読了+反省+実践準備）
- **継続的な実践**: 同時並行で職場でのコーチング実践

### 学習の転移を促進するために

単に各セッションを読むだけでは、実際のコーチング改善につながりません。以下の方法で、学習を行動に転移させることをお勧めします：

1. **実践練習**: 各セッション後に、職場の同僚や部下に対して、学んだスキルを試す
2. **フィードバック収集**: 実践後、相手の反応を観察し、何が上手くいったかを記録
3. **セルフ・リフレクション**: 「次はどうするか」を考え、改善計画を立てる
4. **ピアコーチング**: 同じセッションを学んだ仲間と、実践経験をシェアする

## 関連プログラムとの連携

### CPS（Communication Skill Series）
個別のコミュニケーション技能を深掘りするプログラム。CBSで学ぶコーチングの「傾聴」「質問」「フィードバック」をさらに拡張したスキル開発が可能です。

### SLS（Strategic Leadership Series）
リーダーシップスキルを体系的に学ぶプログラム。CBSで学ぶコーチング思考は、リーダーシップの基盤となります。

### HTP（Human Transformation Program）
組織人としての総合的な人材開発プログラム。CBSはその中の「対人支援」領域の専門化です。

### ICF, IES, ECS（他のコーチング関連シリーズ）
今後実装予定の関連シリーズ。CBSで習得した基礎を発展させ、より高度なコーチングアプローチを学べます。

## 継続的な学習と発展

### アフター・プログラム

CBSプログラム完了後の継続的な発展：
- 定期的な自己評価（月次・四半期）
- スーパービジョン（経験者からのコーチング）
- 実践事例の研究と改善
- 同志とのコミュニティ形成

### 上級コーチング

CBSを修了した後、さらに深い学習へ：
- 特定領域の専門化（リーダーシップコーチング、キャリアコーチングなど）
- コーチング理論の研究
- 認定資格取得（ICF認定コーチなど）
- トレーナー・スーパーバイザーへのキャリアパス
