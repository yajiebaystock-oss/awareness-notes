# awareness.co.jp 動画URL確認ガイド

## 概要

このガイドでは、awareness.co.jp 内の3つの動画の**完全な URL を取得する方法**について、ステップバイステップで説明します。

対象ファイル：
1. **無限大の原則** - URL 部分不完全（XXX/XXXX）
2. **繰の原則** - URL 部分不完全（XXXX）
3. **潜在意識の原則** - URL 未取得（認証が必要）

---

## 前提条件

- awareness.co.jp にログイン済みのブラウザ（Chrome、Firefox、Edge など）
- インターネット接続
- 管理者アカウントまたはアクセス権限がある状態

---

## ステップ1: awareness.co.jp にアクセスする

### 1-1. ブラウザで awareness.co.jp にアクセス

1. **ブラウザのアドレスバー**に以下を入力
   ```
   https://www.awareness.co.jp/
   ```

2. **ログイン画面**が表示された場合：
   - メールアドレス：`brand-all@awareness.co.jp`
   - パスワード：設定しているパスワードを入力
   - **ログイン**をクリック

### 1-2. マイページ > 学習ページへ移動

1. ログイン後、右上のメニューから**「マイページ」**または**「学習」**をクリック
2. 左側サイドメニューから**「動画視聴」**または**「10の能力動画」**を選択

---

## ステップ2: 各動画ページを開く（スクリーンショット位置指示付き）

### 2-1. 無限大の原則

1. **動画一覧ページ**で「無限大の原則」のタイトルを探す
   - リスト内では通常、タイトルや講師名で検索できます
   - **Ctrl+F**（Windows）または**Cmd+F**（Mac）で「無限大」と検索

2. **「無限大の原則」をクリック**して動画ページを開く

3. ページが読み込まれたら**ブラウザのアドレスバーを確認**
   - 表示される URL が次のような形式になっています：
   ```
   https://www.awareness.co.jp/mypage/archives/study-watch/348/[数字]/[数字]/
   ```

4. **スクリーンショット：** ページ左上に「無限大の原則」というタイトルが表示されている場合、その下の講師情報やタイムスタンプ領域を確認します

### 2-2. 繰の原則

1. **動画一覧ページ**に戻る（前ページボタンまたはサイドメニューから）

2. **「繰の原則」のタイトルを探して選択**
   - **Ctrl+F**で「繰」と検索

3. 動画ページが開いたら**アドレスバーの URL をメモ**
   ```
   https://www.awareness.co.jp/mypage/archives/study-watch/348/376/[数字]/
   ```

### 2-3. 潜在意識の原則

1. **動画一覧ページ**に戻る

2. **「潜在意識の原則」のタイトルを探して選択**
   - **Ctrl+F**で「潜在意識」と検索

3. 別ページやダイアログが表示される場合があります
   - **ウィンドウを開く**ボタンや**動画ページを開く**ボタンをクリック

4. 新しいタブで動画ページが開いたら、**アドレスバーの URL をメモ**

## ステップ3: ブラウザから正確な URL をコピーする

### 方法 A: 直接コピー（最も簡単）

1. **アドレスバーをクリック**（URL が表示されている部分）
   
2. **全て選択**：
   - Windows: **Ctrl+A**
   - Mac: **Cmd+A**

3. **コピー**：
   - Windows: **Ctrl+C**
   - Mac: **Cmd+C**

4. **メモ帳やテキストエディタに貼り付け**：
   - Windows: **Ctrl+V**
   - Mac: **Cmd+V**

5. 取得した URL を確認：
   ```
   完全な URL の例：
   https://www.awareness.co.jp/mypage/archives/study-watch/348/376/2024/
   ```

### 方法 B: ブラウザの開発者ツールで確認（確実な方法）

#### Chrome / Edge / Firefox の場合

1. **F12 キー**を押す（またはデベロッパーツールを右クリック > 検査）

2. **Console タブ**を選択

3. 以下を貼り付けて **Enter** キーを押す：
   ```javascript
   console.log(window.location.href);
   ```

4. **結果として表示される完全な URL をコピー**

#### または、より簡単な方法：

1. ページを開いた状態で **F12** を押す
2. **Network タブ**を選択
3. ページをリロード（**F5**）
4. 一番上の行をクリック（ページそのもの）
5. 右パネルの **Headers** で Request URL を確認

## ステップ4: 取得した URL を確認する

### URL フォーマット確認

各動画の URL は以下の形式である必要があります：

| 動画 | 期待される形式 | 例 |
|------|----------------|-----|
| 無限大の原則 | `https://www.awareness.co.jp/mypage/archives/study-watch/348/[3-4桁]/[4桁]/` | `https://www.awareness.co.jp/mypage/archives/study-watch/348/375/2024/` |
| 繰の原則 | `https://www.awareness.co.jp/mypage/archives/study-watch/348/376/[4桁]/` | `https://www.awareness.co.jp/mypage/archives/study-watch/348/376/2024/` |
| 潜在意識の原則 | `https://www.awareness.co.jp/mypage/archives/study-watch/348/[3-4桁]/[4桁]/` | `https://www.awareness.co.jp/mypage/archives/study-watch/348/377/2024/` |

### チェックリスト

- [ ] URL は `https://www.awareness.co.jp` で始まっている
- [ ] URL に `/mypage/archives/study-watch/348/` が含まれている
- [ ] URL の最後に 3 ～ 4 つの数字（年号や ID）がある
- [ ] XXX や XXXX といったプレースホルダーがない
- [ ] 末尾にスラッシュ（/）がある

## ステップ5: 取得した URL をファイルに自動保存する

Python スクリプト（`update_video_urls.py`）を使用して、取得した URL を自動的に Markdown ファイルに保存できます。

### スクリプトの使用方法

#### 準備（初回のみ）

1. **Python 3.7 以上がインストール**されていることを確認
   ```bash
   python --version
   ```

2. **スクリプトを配置**
   ```
   awareness-notes/00_ガイド/update_video_urls.py
   ```

#### 実行方法

1. **PowerShell または コマンドプロンプト**を開く

2. **スクリプトが配置されたディレクトリに移動**
   ```bash
   cd "C:\Users\yajim\OneDrive\デスクトップ\awareness-notes\00_ガイド"
   ```

3. **スクリプトを実行**
   ```bash
   python update_video_urls.py
   ```

#### インタラクティブ入力

スクリプトが起動すると、以下のように各動画の URL を入力するよう求められます：

```
========================================
【10の能力動画】URL 確認・更新ツール
========================================

[1/3] 無限大の原則の URL を入力してください：
URL > 
```

- **取得した URL をコピー＆ペースト**
- **Enter キー**を押す
- 次の動画に進む

#### 処理結果

スクリプト実行後、以下が自動的に更新されます：

1. **【10の能力動画】無限大の原則.md**
   - `source_url:` フィールドが更新
   - YAML フロントマター内の URL が完全な形になる

2. **【10の能力動画】繰の原則.md**
   - 同様に URL が更新

3. **【10の能力動画】潜在意識の原則.md**
   - 同様に URL が更新

## ステップ6: 更新の確認

スクリプト実行後、以下を確認してください：

### Markdown ファイル確認

各ファイルを開いて、以下の部分が更新されたかチェック：

```yaml
source_url: "https://www.awareness.co.jp/mypage/archives/study-watch/348/376/2024"
```

- [ ] URL の XXX や XXXX が数字に置き換わっている
- [ ] URL が完全な形（末尾にスラッシュあり）

### バックアップの確認

スクリプト実行時、元のファイルは自動的にバックアップされます：

```
【10の能力動画】無限大の原則.md.bak
【10の能力動画】繰の原則.md.bak
【10の能力動画】潜在意識の原則.md.bak
```

万が一の場合は、`.bak` ファイルから復旧できます。

## トラブルシューティング

### Q1: ブラウザのアドレスバーに URL が表示されない

**原因:** JavaScript で動的にページが読み込まれている可能性

**解決方法:**
1. ページが完全に読み込まれるまで **5 ～ 10 秒待つ**
2. **F5 キー**でページをリロード
3. **F12 > Console** タブで、上記の JavaScript コマンドを実行

---

### Q2: 「403 Forbidden」エラーが表示される

**原因:** awareness.co.jp にログインしていない、またはアクセス権限がない

**解決方法:**
1. **ログイン状態を確認**（右上のプロフィールアイコンで確認）
2. **必要な場合は再度ログイン**
   - ブラウザキャッシュをクリア：**Ctrl+Shift+Delete** → キャッシュ削除
3. 別のブラウザ（Chrome、Firefox など）で試す
4. **プライベートウィンドウ/シークレットウィンドウ**で試す

---

### Q3: URL が `/348/376/XXXX` という形式で、最後の部分が XXXX のままである

**原因:** ページが完全に読み込まれていない、またはデータが部分的に取得されている

**解決方法:**
1. ページの **右下にある「読み込み完了」** インジケーターを確認
2. **下にスクロール** して、動画プレイヤーが表示されているか確認
3. **ネットワーク接続**を確認（遅い場合は待つ）
4. ブラウザのキャッシュをクリア後、再度アクセス

---

### Q4: Python スクリプトが実行されない

**原因:** Python が環境変数に登録されていない、またはパスが正しくない

**解決方法:**

**Windows の場合:**

```bash
# Python のパスを確認
where python

# または
python -c "import sys; print(sys.executable)"
```

Python が見つからない場合：
- [python.org](https://www.python.org/downloads/) から Python 3.9+ をダウンロード
- インストール時に **「Add Python to PATH」をチェック**して再インストール

**実行方法の代替案:**
```bash
# Python ファイルを右クリック > Python で実行
# または
python.exe "C:\Users\yajim\OneDrive\デスクトップ\awareness-notes\00_ガイド\update_video_urls.py"
```

---

### Q5: スクリプト実行中に「permission denied」エラー

**原因:** ファイルへの書き込み権限がない

**解決方法:**
1. **エクスプローラー**で該当ファイルを右クリック
2. **プロパティ** → **詳細** → **セキュリティ**
3. **フルコントロール**権限を確認、ない場合は付与
4. または、**管理者として実行** でコマンドプロンプトを開く

---

### Q6: 「URL が無効な形式です」というエラーメッセージ

**原因:** 入力した URL が HTTP（S）で始まっていない、または形式が違う

**解決方法:**
1. **完全な URL をコピー** （前のセクション「ステップ3」参照）
2. URL が `https://` で始まっていることを確認
3. アドレスバー全体をコピーすることを確認（余分なスペースなし）

---

### Q7: ファイルが更新されない

**原因:** ファイルがロックされている（別のアプリで開いている）、または権限がない

**解決方法:**
1. **Markdown エディタやテキストエディタ**でファイルが開いている場合は**閉じる**
2. VS Code など IDE を使用している場合は、**一度ファイルをクローズして再度開く**
3. スクリプト実行後、エディタで **「外部変更を再読み込み」** を実行

## よくあるご質問（FAQ）

### Q: URL が頻繁に変わることはありますか？

**A:** 通常、一度確定した動画の URL は変わりません。ただし、サーバー側のリダイレクトやメンテナンスがある場合は変更される可能性があります。

---

### Q: 複数の PC で同じ URL を使用できますか？

**A:** awareness.co.jp が認証を必要としているため、**同一アカウントでログイン**していれば利用可能です。ただし、個別のセッションは別になります。

---

### Q: セキュリティ上の懸念はありますか？

**A:** 
- URL 自体には機密情報は含まれていません
- スクリプトはローカル環境でのみ実行され、インターネットに送信されません
- Markdown ファイルはバージョン管理対象外にすることをお勧めします
