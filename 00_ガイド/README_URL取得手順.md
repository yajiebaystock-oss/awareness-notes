## 🔧 ツール・スクリプト一覧

### 1. **update_video_urls.py** - URL 更新スクリプト

**用途:** 取得した URL を対話的に入力して、Markdown ファイルを自動更新

**実行方法:**
```powershell
python update_video_urls.py
```

**処理内容:**
- 現在の URL 状態を表示
- 各動画の新しい URL を対話的に入力させる
- 入力された URL を検証（形式チェック）
- バックアップを作成
- Markdown ファイルの `source_url` フィールドを更新

**出力:**
- ✓ 更新されたファイルの一覧
- ⚠️ エラーメッセージ
- バックアップファイル（`.bak`）

---

### 2. **batch_check_urls.py** - URL 状態確認スクリプト

**用途:** 3つのファイルの URL 状態を一括確認（更新前・更新後の確認用）

**実行方法:**
```powershell
python batch_check_urls.py
```

**確認内容:**
- ✓ URL が完全な形か
- ⚠️ URL が不完全（プレースホルダー含む）か
- ✗ URL が取得できていないか

**出力例:**
```
【URL 確認結果】

✓ 無限大の原則
  状態: 完全
  URL: https://www.awareness.co.jp/mypage/archives/study-watch/348/375/2024/

⚠ 繰の原則
  状態: 不完全
  URL: https://www.awareness.co.jp/mypage/archives/study-watch/348/376/XXXX
  問題: URL に プレースホルダー（XXX/XXXX）が含まれています

✗ 潜在意識の原則
  状態: 未取得
  URL: Unable to extract - authentication required
  問題: 認証が必要で URL が取得できません
```

### Q: どのガイドから始めればいい？

### Q: Python がインストールされていない場合は？

**A:** Python をインストールしてください：

1. [python.org](https://www.python.org/downloads/) から Python 3.9 以上をダウンロード
2. インストール時に **「Add Python to PATH」をチェック**
3. インストール後、ターミナルを再起動

確認方法：
```powershell
python --version
```

---

### Q: エラーが出た場合は？

**A:** 詳細は **クイックレファレンス_URL確認.md** の「トラブルシューティング」セクションを参照してください。

よくあるエラー：
- `ModuleNotFoundError` → Python パスの問題
- `PermissionError` → ファイル権限の問題
- `UnicodeEncodeError` → 文字コードの問題

---

### Q: スクリプトが対応する URL フォーマットは？

**A:** 以下の形式です：

```
https://www.awareness.co.jp/mypage/archives/study-watch/348/[ID]/[年度]/
                                                            ↑     ↑
                                                         3-4桁  4桁以上
```

例：
- `https://www.awareness.co.jp/mypage/archives/study-watch/348/375/2024/`
- `https://www.awareness.co.jp/mypage/archives/study-watch/348/376/2024/`

---

### Q: ファイルが更新されない場合は？

**A:** 以下を確認してください：

1. **Markdown エディタが開いていないか**
   - 開いている場合は閉じてからスクリプト実行

2. **ファイルに書き込み権限があるか**
   - 右クリック → プロパティ → セキュリティ で確認

3. **別のアプリケーションが使用していないか**
   - VS Code, Sublime Text など IDE が開いている場合は閉じる
