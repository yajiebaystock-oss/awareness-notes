↓
指摘事項対応（フィードバックループ）
↓
最終承認

| **品質改善提案** | 品質向上方策の提案 | 随時 | 随時 |

### チェック項目リスト

#### 迅速チェック（全ファイル）

- [ ] ファイルサイズ: 800字以上8,000字以下
- [ ] Frontmatter: 全7項目完備
- [ ] Markdown 見出し: ## / ### 正しく使用
- [ ] Frontmatter フォーマット: YAML 正しい構文
- [ ] タグ数: 3～10個
- [ ] ファイル命名: 規則に適合

**チェック時間**: 3～5分/ファイル

#### 詳細チェック（サンプル20%）

- [ ] 要約の完全性: 3～5行
- [ ] ポイント数: 8～12項目
- [ ] 内容の一貫性: 論理的流れ
- [ ] 日本語の正確性: タイプミス等
- [ ] リンク確認: 全リンク有効
- [ ] メタデータと本文の一貫性

**チェック時間**: 15～20分/ファイル
