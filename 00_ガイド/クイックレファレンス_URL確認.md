# 【クイックレファレンス】URL確認 - 5分版

## TL;DR（30秒版）

```
1. awareness.co.jp にログイン
2. 各動画ページを開く
3. アドレスバーから URL をコピー
4. python update_video_urls.py を実行 → URL をペースト
```

## コマンド一覧

### 対象ファイルの URL 状態を確認（実行前）

```bash
python batch_check_urls.py
```

**出力例:**
```
✓ 無限大の原則        → 完全
⚠ 繰の原則           → 不完全（XXX/XXXX が含まれている）
✗ 潜在意識の原則     → 未取得（認証が必要）
```

### URL を入力・更新

```bash
python update_video_urls.py
```

**対話的に入力:**
```
[1/3] 無限大の原則
現在の URL: https://www.awareness.co.jp/mypage/archives/study-watch/348/XXX/XXXX
新しい URL を入力 > https://www.awareness.co.jp/mypage/archives/study-watch/348/375/2024/
```

### 更新後に状態を再確認

```bash
python batch_check_urls.py
```

## URL フォーマット チェックリスト

| チェック項目 | 例 | ✓/✗ |
|-----------|------|------|
| https:// で始まっている | `https://www.awareness.co.jp/...` | ✓ |
| awareness.co.jp を含む | `.awareness.co.jp/` | ✓ |
| /mypage/archives/study-watch/348/ を含む | `/348/` | ✓ |
| XXX や XXXX がない | `...376/2024/` | ✓ |
| 最後に 4 桁の数字がある | `.../2024/` | ✓ |
| 末尾がスラッシュ | `.../2024/` | ✓ |

完全な例:
```
https://www.awareness.co.jp/mypage/archives/study-watch/348/376/2024/
                                                            ↑   ↑
                                                          ID  年度
```

## よくある問題と解決

| 問題 | 原因 | 解決法 |
|-----|------|------|
| `403 Forbidden` | ログインしていない | awareness.co.jp に再度ログイン |
| URL が表示されない | JS 読み込み中 | 5～10秒待つ → F5 リロード |
| Python が見つからない | PATH に登録されていない | python.org から再インストール |
| ファイルが更新されない | 別アプリで開いている | エディタを閉じてからスクリプト実行 |

## URL 取得のコツ

### ✅ 確実な方法

```
1. awareness.co.jp にログイン（右上でアカウント名を確認）
2. 【無限大の原則】などのタイトルをクリック
3. ページが完全に読み込まれるまで待つ（画像やプレイヤーが表示される）
4. アドレスバーをクリック → Ctrl+A → Ctrl+C（コピー）
5. メモ帳などに貼り付けて確認
6. スクリプトにペースト
```

### ❌ 避けるべき操作

- ❌ 不完全なページでコピー（プレイヤー読み込み中）
- ❌ プライベートウィンドウ使用（セッション別）
- ❌ キャッシュなしでの閲覧（ページ表示遅延）
