# 10の能力動画 - ビデオダウンロード & 文字起こしスキル

> このスキルを使用するには、Obsidian の Claude Code プラグインで `/ability-transcribe` コマンドを入力してください。

## 説明

10の能力動画（Brightcove ビデオ）をダウンロードして、Whisper で自動文字起こしし、Claude で要約を生成。Obsidian ファイルに保存するスキル。

**対応フォルダ**: `01_講義/10の能力動画/`

**処理フロー**:
1. Brightcove ビデオプレイヤーから MP4 をダウンロード
2. ffmpeg で m4a 音声に変換
3. Whisper（日本語）で文字起こし
4. Claude で要約自動生成
5. Markdown + frontmatter で Obsidian に保存

---

## コマンド一覧

### `/ability-transcribe-download`
**Brightcove ビデオをダウンロード**

使用方法:
```
/ability-transcribe-download https://players.brightcove.net/6054371505001/... episode_name
```

入力:
- `video_url`: Brightcove プレイヤー URL
- `episode_name`: エピソード名（ファイル名になる）

出力:
- MP4 ファイル（`temp/` フォルダ）

---

### `/ability-transcribe-process`
**ダウンロード済みのビデオを文字起こし → 要約 → Obsidian 保存**

使用方法:
```
/ability-transcribe-process temp/video_6398801618112.mp4 "バランスの原則"
```

入力:
- `mp4_file`: ダウンロード済みの MP4 ファイルパス
- `title`: エピソード名（Markdown タイトルになる）

処理内容:
1. ffmpeg で m4a に変換
2. Whisper で日本語文字起こし
3. Claude で要約生成
4. Markdown + frontmatter 生成
5. Obsidian フォルダに自動保存

出力:
- `01_講義/10の能力動画/【10の能力動画】[タイトル].md`

---

### `/ability-transcribe-all`
**すべてのビデオを一括処理**

使用方法:
```
/ability-transcribe-all
```

処理対象（Brightcove videoId 確認済み）:
- バランスの原則 (6398801618112)
- 原因と結果の原則 (6363582566112)
- 振動と波動の原則 (6363631477112)
- 道徳（決断）の原則 (6363583239112)
- 表裏一体の原則 (6363584192112)
- 法則・原則の定義 (6363584189112)

## 実装済み機能

### 1️⃣ Whisper 文字起こし

```python
from src.transcribe import transcribe_japanese
from pathlib import Path

# 日本語の文字起こし
transcript = transcribe_japanese(
    Path("audio.m4a"),
    model_name="base"
)
```

### 2️⃣ Claude 要約生成

```python
from src.summarize import generate_summary

summary = generate_summary(
    transcript,
    summarizer="claude",
    claude_model="claude-3-5-sonnet-20241022",
    max_chars=12000
)
```

### 3️⃣ Markdown 自動生成

```python
from src.markdown_writer import build_markdown, save_markdown

markdown = build_markdown(
    title="バランスの原則",
    source_url="https://www.awareness.co.jp/...",
    summary=summary,
    transcript=transcript,
    category="10の能力動画"
)
saved_path = save_markdown(markdown, vault_path, title)
```

## トラブルシューティング

### ffmpeg が見つからない
```bash
# ffmpeg をインストール（Windows）
choco install ffmpeg
# または Scoop
scoop install ffmpeg
```

### Whisper がない
```bash
pip install openai-whisper
```

### ビデオダウンロード失敗
- yt-dlp の更新: `pip install --upgrade yt-dlp`
- Brightcove URL の確認
- ネットワーク接続の確認
